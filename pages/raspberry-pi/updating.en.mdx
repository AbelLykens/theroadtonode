# Updating

<Callout emoji="&#x2139;">
        Time: 2 minutes
</Callout>

Since we are now running a fresh install from the SSD, we do not have the updates from just now. So SSH into the Pi and run the following again.

```bash
sudo apt update
```

```bash
sudo apt upgrade -y
```

```bash
sudo reboot now
```
